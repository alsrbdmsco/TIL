
-----------
> 줄바꿈

* 3칸 이상 띄어쓰기 하면 줄이 바뀐다. _ = 띄어쓰기
```md
*가나다라마바사.___
 아자차카타파하.
```

*가나다라마바사.   
아자차카타파하.

> 정리

마크다운은 기본 문법만 알고있으면 손쉽게 작성이 가능하다. 다양한 도구와 플랫폼에서 지원하고 있어서 많이 사용되고 있다.
