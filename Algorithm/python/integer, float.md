# 정수와 실수

> 정수

- 정수끼리 더하거나 곱하거나 빼면 정수
- 정수끼리 나누면 실수가 나올 수 있으나, 나눗샘의 몫만을 구하려면 //연산자를 이용
```
a = 5//3 #계산결과 a=1
```
- 실수를 정수로 바꾸려면 int를 이용
- a=int(5.4)라고 하면 a는 5를 값으로 가지게 된다.

------------------
> 실수

- 소숫점을 표시할 수 있는 숫자
- 계산에 있어서 완벽한 정확성은 없다.
```
0.1+0.1+0.1 == 0.3 #FALSE
```
- 정수를 실수로 바꾸려면 float를 사용

- a=float(8)라고 하면 a는 8.0을 값으로 가지게 된다.
