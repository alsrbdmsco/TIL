# Variable(변수)

> 변수 선언 방법
```
변수의 이름 = 값
```

- 예) 변수 name에 이름을 넣고 출력하세요.

```python
name='김민규'
print('제 이름은',name,'입니다.')
```
- 결과
```
제 이름은 김민규 입니다.
```
---------------
> 주석 #
```
주석은 코드 옆에 설명을 할 때 또는 코드를 임시로 작동하지 않게 한다.
그 때는 코드 옆에 # 을 붙이면 주석처리가 된다. 많은 줄을 주석 처리 하고 싶을 때는
""" 를 주석 시작 부분과 끝 부분에 써주면 된다.
```
- 한줄 주석
```
# print('제 이름은',name,'입니다.')
```
- 여러줄 주석
```
"""
# print('제 이름은',name,'입니다.')
# print('제 이름은',name,'입니다.')
# print('제 이름은',name,'입니다.')
"""
```
